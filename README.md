# Multi-Year Groundwater Level and Water Quality Prediction Using Temporal Fusion Transformer (TFT)

## Overview

This project presents a deep learning-based framework for predicting groundwater levels and water quality using the Temporal Fusion Transformer (TFT). It analyzes historical groundwater data collected from multiple monitoring stations across India and forecasts future groundwater conditions to support sustainable water resource management.

The application also provides an interactive web interface where users can explore historical trends and visualize future predictions.

## Installation

Clone the repository:

```bash
git clone https://github.com/Lakshithap21/Multi-Year-Groundwater-Level-and-Water-Quality-Prediction-Using-Temporal-Fusion-Transformer.git
```

Move to the project directory:

```bash
cd Multi-Year-Groundwater-Level-and-Water-Quality-Prediction-Using-Temporal-Fusion-Transformer
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```
